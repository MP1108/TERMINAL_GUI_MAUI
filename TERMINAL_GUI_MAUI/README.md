# TERMINAL_GUI_MAUI - Product Management App

A mobile application for product management built with .NET MAUI.

## Features
- ✅ Add products
- ✅ Edit products  
- ✅ Delete products
- ✅ Export to CSV
- ✅ Product categories
- ✅ Operation logging

## Technologies
- .NET MAUI
- C#
- XAML
- MVVM Pattern

## Installation & Run
1. Clone the repository
2. Open project in Visual Studio or Rider
3. Run on Android or iOS emulator
